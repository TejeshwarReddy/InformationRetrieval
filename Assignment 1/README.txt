Guidelines and working of our project. 

1) Our Project is in ‘InformationRetrieval’ folder.
2) Building the project opens a GUI where the user can type the query.
3) The user can also selct the number of documents to be retrieved.
4) The documents are retrieved in the form of a list.
5) The user can Right-Click on the List Item and click on show to view the content of the document.
